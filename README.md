## What I'm Building

- 🤖 **[Agent Infrastructure](https://github.com/acoomes/agent-infrastructure)** — Three services for the autonomous AI agent economy: shared memory, API marketplace with x402 payments, and bounded delegation identity. Think of it as plumbing for agents that need to collaborate, spend money, and not go rogue.

- 📈 **[Dumbtrader](https://github.com/acoomes/dumbtrader)** — A futures trading bot built on the ProjectX API. Define strategies with YAML or Pine Script and let it loose on paper or live accounts. The name is aspirational honesty.

- 🎲 **[DiceVZombz](https://github.com/acoomes/dicevzombz)** — A dice-rolling zombie survival game built with vanilla JS. Because sometimes you just need to roll dice and fight the undead.

- ☀️ **[Morning Download](https://github.com/acoomes/morning-download)** — Daily news and AI briefing, auto-generated by Claude at 6:00 AM PT. Because reading the news yourself is so 2024.
